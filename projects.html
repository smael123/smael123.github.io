Personality Patterns among US Politicians

This was my Senior Project. I wanted to find personality patterns among US politicians (hence the title) using Twitter data. I was to gather the tweets, get personality percentile scores for each politician based on their concacted tweets.

Gathering Tweets

First I got an excel file with the names of all the people in the House of Representatives from their website

I used the python-twitter API and the Python programming language. Using this API I first gathered the names of the people in the House of Representatives. It was from a twitter list by Twitter Government but a I write this the link I used is dead. I used the GET lists/list part of the API

Using openpyxl, I inserted the twitter handle (screen name) to the corresponding name based on wether or not the real name matched. It almost worked fully. Some had middle names in the government file that did not match the Twitter name. And some used suffixes like Jr. that were again not in the Twitter name. For those cases I manually entered the data.

When you gather tweets from twitter you can save the result as a JSON file. I saved the JSON files in a separate folder with each file being named after the politician. However this was approach was not very good. There are special chracters in some of the file names and I had trouble reading the files. Not only that but this approach is not portable since other operating systems have different file name requirements. I got it to work but in the future I think I would have either one big JSON file, or index each politican in both the excel file and the file name.

Getting the Personality Scores

To get the personality percentile scores I used IBM Watson's Personality Insights API (PI). I sent the API the concacted tweet and it returns a personality scores for that concacted tweet. You could use any piece of text. It might be fun to get the personality scores for movie characters. The scores include big 5 values, facets, needs, and values. Having these scores I was finally able to analyze the tweets.

Analyzing the Personality Scores

From the start it was decided that we would cluster that data. For this I used R, a vector language used by data scientists. I used the k-means algorithm built in to the base package of R to cluster the data.

After this I analyzed the data by making box plots using Excel. This should've been done before clustering You should have a look at the data before clustering to see attributes that have large variacnces.
